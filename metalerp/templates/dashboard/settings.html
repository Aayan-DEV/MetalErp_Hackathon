How to connect
  1. Go to the Google Cloud Console and enable the Vertex AI API for your project
  2. Navigate to IAM & Admin → Service Accounts
  3. Click Create Service Account, give it a name, and grant the Vertex AI User role
  4. Click the service account, go to the Keys tab, then Add Key → Create new key → JSON
  5. Copy your Project ID from the top of the console (or from the downloaded JSON's project_id field)
  6. Paste the Project ID and the full JSON key contents below
